The path filter to restrict search to specific directories or a file. Use it for the single allowed retry search after the first result identified a relevant area. Set it to the directory of the best first hit instead of searching the whole repository again. Must be a path relative to the project root. If not provided, the whole project is searched. Examples: "path/to/module", "path/to/module/submodule", "path/to/file/example.kt".
